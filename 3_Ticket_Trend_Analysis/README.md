# 客服工单趋势分析 Dashboard

基于 **Streamlit** 的交互式工单数据分析工具，帮助客服主管快速发现趋势变化和异常信号。

## 项目概述

客服主管每天要关注大量工单数据，需要快速了解：哪些问题最近变多了？哪些异常需要关注？本工具自动从工单数据中发现趋势和异常，帮主管把精力放在真正需要关注的事情上。

## 技术栈

| 层面 | 技术 |
|------|------|
| 框架 | Streamlit |
| 数据处理 | Pandas + NumPy |
| 可视化 | Matplotlib + Seaborn（中文适配） |
| 异常检测 | Z-score(2σ) + 7日滚动平均 + IQR 箱线图（三级组合） |

## 分析维度设计

### 1. 总览 — 全局仪表盘
- KPI 卡片：总工单量、平均满意度、平均处理时长、未解决率、高优先级占比
- 分类/优先级环形图
- **决策价值**：一眼掌握全局状况，快速评估团队整体表现

### 2. 时间趋势 — 发现峰谷
- 每日工单量折线图（含 7 日滚动平均线）
- 各分类逐日堆叠面积图
- 高/中/低优先级分面趋势图
- **决策价值**：发现工单量突变和趋势拐点，辅助排班和资源调度

### 3. 分类分析 — 识别集中领域
- 分类工单量横向排行柱状图
- 分类 × 优先级交叉热力图
- 各分类满意度对比（含整体均值基准线）
- **决策价值**：遵循帕累托法则，识别 20% 问题类型贡献 80% 工单量，指导知识库建设和 AI 自动应答

### 4. 处理效率 — 定位瓶颈
- 各分类平均处理时长横向柱状图
- 各优先级处理时长箱线图
- 处理时长 vs 满意度散点图 + 回归线
- **决策价值**：发现流程阻塞点，量化处理效率对满意度的影响

### 5. 满意度分析 — 定位洼地
- 满意度分布直方图 + KDE + 均值线
- 各渠道满意度分组柱状图（含低分率）
- **决策价值**：定位满意度最低的分类和渠道，发现服务质量问题

### 6. 异常检测 — 自动预警（核心）
采用 **三级递进检测 + 额外业务规则** 引擎：

| 检测器 | 方法 | 阈值 | 角色 |
|--------|------|------|------|
| 第一级 | Z-score | 2σ（超出均值 ± 2 倍标准差） | 主检测器，捕获显著偏离 |
| 第二级 | 7 日滚动平均偏离 | 偏离 > 30% | 趋势检测器，捕获渐进恶化 |
| 第三级 | IQR 箱线图 | 1.5×IQR（超出四分位距 1.5 倍） | 鲁棒校验器，不依赖正态假设 |

**综合判定**：≥2 级命中 → 确认异常；1 级 → 疑似异常

**额外业务规则**：
- 连续 ≥3 天满意度下降 → 持续恶化预警
- 某分类未解决率 > 整体均值 + 15% → 解决能力预警
- 某分类单日占比 Z-score > 2σ 且 > 30% → 分类突增预警

**决策价值**：直接呈现需要关注的问题，每个异常附带判断依据、涉及工单 ID 和行动建议，减少人工排查时间

### 7. 渠道分析 — 优化资源配置
- 渠道占比环形图
- 渠道 × 分类分组柱状图
- 渠道综合对比表（满意度、处理时长、未解决率）
- **决策价值**：针对性改进各渠道服务质量，优化渠道资源配置

## 基于实际数据的关键发现

> 以下分析基于 50 条工单数据（2024-06-01 ~ 2024-06-11）自动生成。

### 工单分布
- **支付问题**（16条，32%）和**退款退货**（13条，26%）合计占比 58%，符合帕累托法则
- 高优先级工单占比 **62%**，整体服务压力较大
- 在线渠道占 **80%**，电话占 20%（数据中无邮件工单）

### 处理效率
- 处理最快：**商品咨询**（平均 0.9h），标准化问答，适合 AI 自动应答
- 处理最慢：**退款退货**（平均 45.2h），涉及退款流程和多部门协作
- 高优先级工单处理时长中位数 6h，但存在极端长尾（最长 120h）

### 满意度
- 整体满意度均值 **2.36/5**，偏低（行业基准通常 3.5+）
- 满意度最高：**商品咨询**（4.2/5），最快的问题得到最好的评价
- 满意度最低：**投诉**（1.0/5）和**退款退货**（2.0/5）
- 处理时长与满意度呈**显著负相关**：时长越长，满意度越低

### 异常信号（自动检测）
经三级检测引擎识别：
- 🔴 **满意度连续下降**（6/4→6/6，3天）：期间 14 条工单中 6 条低分，集中在退款退货和支付问题
- 🔴 **满意度连续下降**（6/9→6/11，3天）：期间 16 条工单中 10 条低分，支付问题爆发（7条），5条未解决
- 🟡 **退款退货未解决率异常**：38.5%（5条未解决 / 13条总量），远超整体 16%，需重点排查

## 安装与运行

### 环境要求
- Python 3.9+
- pip

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行 Dashboard
```bash
streamlit run app.py
```

浏览器访问 `http://localhost:8501` 即可查看。

### 使用指南
1. 左侧边栏**全局筛选器**：按日期、分类、优先级、渠道筛选数据，所有图表联动更新
2. 左侧边栏**导航切换**：选择不同分析模块查看对应图表
3. 每个图表下方提供 **导出 PNG** 按钮
4. 异常检测模块提供**异常卡片**（含判断依据、涉及工单、行动建议）+ **导出 CSV**
